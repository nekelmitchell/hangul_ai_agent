# Hangul Tutor AI Agent

## Overview

Hangul Tutor AI Agent is an adaptive educational AI system designed to help beginners learn the Korean alphabet (Hangul) through personalized lessons, quizzes, conversation practice, and memory-based tutoring.

Unlike a traditional quiz application, the agent remembers previous mistakes, identifies weak letters, tracks learning habits, and uses AI-generated feedback to personalize future study sessions.

This project was built to explore AI-agent concepts such as memory systems, retrieval, personalization, adaptive learning, and large language model reasoning.


## Features

* Memory System
* Adaptive Learning
* Conversational Korean Practice
* AI-Powered Tutoring

## Installation

1. Clone the repository

git clone [https://github.com/yourusername/hangul-ai-agent.git](https://github.com/nekelmitchell/hangul_ai_agent.git)

2. Navigate into the project folder

cd hangul-tutor-ai-agent

3. Install dependencies

pip install -r requirements.txt

4. Create a .env file

OPENAI_API_KEY=your_api_key_here

5. Run the program

python main.py
